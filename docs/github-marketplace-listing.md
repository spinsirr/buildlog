## Description (Markdown)

### Turn shipping into marketing

You push code every day. BuildLog turns those pushes into social media posts — automatically.

**How it works:**

1. **Connect your repos** — Install the GitHub App. BuildLog monitors pushes, merged PRs, and releases.
2. **AI reads the diff** — Not just commit messages. BuildLog reads your actual code changes and understands what you shipped.
3. **Publish everywhere** — Platform-optimized posts: concise for Twitter/X, long-form for LinkedIn, native for Bluesky. Review in your dashboard or auto-publish.

**What makes BuildLog different:**

- **Code-aware AI** — reads diffs, not just commit messages. It knows you shipped a billing dashboard, not just "updated 4 files."
- **Platform-optimized** — each platform gets content tailored to its format and audience. No more copy-pasting a tweet into LinkedIn.
- **Zero writing required** — push code, get posts. Edit if you want, but you don't have to.
- **Public changelog** — every user gets a `/changelog/username` page showing their shipping history.

**Built for:**
- Indie hackers building in public
- Founders who ship fast but forget to talk about it
- Dev teams that want their engineering work to drive marketing

**Pricing:**
- **Free** — 5 posts/month, 1 repo, 1 platform (LinkedIn or Bluesky)
- **Pro ($19.90/mo)** — Unlimited posts, unlimited repos, all platforms including Twitter/X, auto-publish
